Why CMacs?

Always Insert Mode

Like nano, just open a file and start typing. No mode switching, no learning curve. But with real power when you need it.

Tab-Completed Commands

Press ESC and type. Commands are organized by category with incremental completion. Discover features as you type.

Project Management

Define your project once. Jump between source files with a keystroke. Kick off builds from the editor.

Syntax Highlighting

Full 24-bit RGB color support on modern terminals. C, C++, Python, Markdown, and more.

Split Screens

View two files side by side. Compare code, reference documentation, or edit related files together.

Zero Dependencies

No Boost, no autoconf, no cmake, no package managers. Just make. Builds in seconds.

In Action

Project and buffer management

Integrated build with error navigation

Runs Everywhere

The same codebase compiles on your M3 Mac and on a 1993 SPARCstation. No compromises on modern systems, no abandoning vintage hardware.

Quick Start

# Clone the repositories
mkdir -p ~/dev/cx/cx_apps && cd ~/dev/cx
git clone https://github.com/toddvernon/cx.git
cd cx_apps
git clone https://github.com/toddvernon/cm.git

# Build
cd ~/dev/cx/cx && make
cd ~/dev/cx/cx_apps/cm && make

# Run
./darwin_arm64/cm myfile.cpp
